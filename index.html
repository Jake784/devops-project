Arquitectura de servicios

Amazon Web Services

EC2 t2.micro en us-east-1. VPC vpc-03ceaae con Security Group configurado para puertos 22, 80 y 8080.

Docker Swarm

Orquestación con 4 contenedores: 2 réplicas web (Nginx), 1 API (FastAPI) y 1 base de datos (PostgreSQL).

GitHub Actions CI/CD

Pipeline de 5 etapas: checkout → build → export → SCP → deploy. Se activa en cada push a main.

PostgreSQL + FastAPI

API REST con endpoints CRUD completos. Base de datos con tablas users y tasks con autenticación por sesión.
